# Floor Plan Designer - Table Layout Tool

An interactive web-based tool for designing floor plans with tables of different sizes (4ft, 6ft, and 8ft).

## Features

- **Customizable Room Dimensions**: Set the length and width of your room in feet
- **Multiple Table Sizes**: Add tables of 4ft, 6ft, and 8ft sizes
- **Drag and Drop**: Easily move tables around the floor plan
- **Grid System**: Visual grid to help with alignment
- **Grid Snapping**: Optional grid snapping for precise placement
- **Dimension Display**: Show/hide room and table dimensions
- **Table Management**: Delete individual tables or clear all at once
- **Real-time Count**: See how many tables of each size you've placed

## How to Use

1. **Set Room Dimensions**: 
   - Enter the length and width of your room in feet
   - Click "Update Room Size" to apply changes

2. **Add Tables**:
   - Click on "Add 4ft Table", "Add 6ft Table", or "Add 8ft Table"
   - Tables will appear in the center of the room

3. **Move Tables**:
   - Click and drag any table to reposition it
   - Tables will stay within the room boundaries

4. **Delete Tables**:
   - Hover over a table to see the delete button (×)
   - Click the × button to remove a table

5. **Tools**:
   - **Toggle Grid Snap**: Enable/disable snapping to grid for precise alignment
   - **Show Grid**: Toggle the visual grid on/off
   - **Show Dimensions**: Toggle dimension labels on/off
   - **Clear All Tables**: Remove all tables from the floor plan
